# ChatApp
ChatApp is an application built using Java, Typescript, HTML, and CSS in Spring Boot using technologies such as Websockets in order to create a platform where users can message each other and make groups.

## Installation
Project uses Java version 17, and for installation you will need NPM and Angular installed.
You'll need to change the following:
```properties
spring.datasource.url = jdbc:mysql://your_db_url
spring.datasource.username = your_db_username
spring.datasource.password = your_db_password
```
## Usage
The app is to be used for chatting between people you have contact with, you can search for users on the system or you can invite them in a group through a link.
